Best Double Cuts

This is a visualizer webpage for the Leetcode problem. To increase the fun I will refer to the subarrays left as Red and the right as Blue.

A default input is provided but replace it from the text input field to use your own. When you change the Red and Blue counters you are modifying how many elements will belong to each of their subarrays. The middle array is shown in an off-white shade. At least 1 number must be of each colour which the input enforces.

You will be shown the array in table form with the number along with it's index. These of course, also colour coded. Along with that are the sums of each color and whether the two conditions are independently passing with flying colours.
